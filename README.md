Sistema de Controle de Estoque em Java

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO) e lógica de programação, simulando um sistema simples de controle de estoque via terminal.

🚀 Funcionalidades
📦 Cadastrar produtos
📋 Listar produtos cadastrados
🔄 Atualizar quantidade de produtos
❌ Remover produtos do estoque
🧠 Validação de dados e controle de limites

🛠️ Tecnologias utilizadas
Java
Programação Orientada a Objetos (POO)
Estruturas de controle (if, switch, while, for)
Arrays
Entrada de dados com Scanner
